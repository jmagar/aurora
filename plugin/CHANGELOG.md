# Changelog

All notable changes to the `aurora` plugin are documented here.

## [0.1.0] - 2026-07-16

### Added

- Initial plugin scaffold
- `aurora` skill (copied from the `vibin` plugin) — guidance for
  building Aurora-styled React/Next.js UI with the `@aurora` shadcn registry and
  `var(--aurora-*)` tokens.

The plugin version is independent from the Aurora registry release. It packages
the canonical Aurora skill and references; registry components continue to use
the repository's release/versioning contract.
