# Aurora Shell Theme Pack

Shadcn installs this file into the current project. Source shell theme files live in the Aurora repository under:

- themes/shell/p10k/aurora-p10k.zsh
- themes/shell/statusline/statusline-aurora.sh
- themes/shell/bat/Aurora.tmTheme
- themes/shell/mc/aurora.ini
- themes/shell/nano/nanorc
- themes/shell/zsh/

Shadcn's `~` target resolves to the project root, not the user's real home directory. Copy these files into your shell configuration deliberately after reviewing them.
